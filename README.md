# Bibliotrack

Registro personal de lecturas. Guarda los libros que has leído cada año, con portada, puntuación y reseña — como Letterboxd pero para libros.

## Funcionalidades

- Buscar libros por título o autor (vía Open Library)
- Portadas automáticas
- Puntuación de 1 a 5 estrellas
- Reseña personal
- Estados: Leído, Leyendo, Quiero leer
- Navegación por año
- Los datos se guardan en el navegador (localStorage)

## Uso

Abre `index.html` en el navegador, o visita la versión online.

No requiere instalación ni backend.
